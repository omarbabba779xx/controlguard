# Roadmap

## Current stage

`controlguard` is at the "advanced portfolio / pre-product" stage.

The current goal is to improve credibility, repeatability, and breadth without losing the strict evidence model.

## Near-term

- validate Microsoft Graph connector on a real tenant
- validate Okta connector on a real tenant
- publish GitHub release notes for `v0.1.0`
- add repository issue backlog

## Next expansion

- AWS posture checks

- GCP posture checks
- more Linux security controls
- TLS and certificate validation

## Product polish

- historical scan snapshots
- richer executive dashboards
- optional trend analysis
- signed release artifacts

## Long-term

- multi-tenant usage model
- scheduled scans
- policy packs
- custom framework mapping packs
